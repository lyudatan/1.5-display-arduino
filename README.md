# 1.5-display-arduino

Test For 1.8" TFT LCD with Arduino

Download the ST7735 Libraries from https://github.com/adafruit/Adafruit-ST7735-Library

Connections

1 RST - D8  
2 CS  - D10  
3 DC  - D9  
4 DIN - D11  
5 CLK - D13  
6 VCC - +5V  
7 BL  - +5V  
8 GND - GND
